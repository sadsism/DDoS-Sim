#### A project by sadsism



# Features
### With DDoS-Sim tool, you can:
- Take a view in a fake DDoS Panel 
- Learn about ports & website ips

Anyone may use this tool for educational purposes.

# Installation Guide
## Requirements: 
- Linux OS / Mac OS
- Bash installed ( Usually pre-installed ).

## Installation Guide
#### Clone repository:
```bash
git clone https://github.com/sadsism/DDoS-Sim
cd DDoS-Sim
```
#### Make script excecutable:
```bash
chmod +x ddos-sim.sh
```
#### Run script:
```bash
./ddos-sim.sh
```
## Controls
- CTRL + C: Kill running simulation
- ENTER: Return to main menu
- BACKSPACE: Exit
